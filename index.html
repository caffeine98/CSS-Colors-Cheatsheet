CSS Colors Cheat Sheet

Named Colors

Named colors consist of colors in english. These consist of simple color names such as red, green and blue, or names with modifiers in them such as darkblue, forestgreen and lightseagreen.

        p {
          color: lightseagreen;
        }

Hexadecimal

A hex color begins with a hash character (#) which is followed by three or six characters. The characters represent values for red, blue and green. For example, the color brown can be represnted in hexadecimal as #a52a2a.

        h2 {
          color: #a52a2a;
        }

RGB Colors

Colors can be represnted as RGB values instead of hexadecimal, where three values for red, green, and blue, are used to create the color. The values range from 0 to 255

        header {
          background-color: rgb(143, 188, 143);
        }

HSL Colors

HSL is a different color scheme that uses hue, saturation and lighting to determine the color. The first number represents the degree of the hue, and can be between 0 and 360. The second and third numbers are percentages representing saturation and lightness respectively.

        footer {
          background-color: hsl(120, 60%, 70%);
        }
